Questions

Web Art Installation October 2016

Questions is a web art installation where a program poses questions and searches for the answers on YouTube. Try it here.

A self-reflexive internet

Questions is my attempt to make the viewer watch the internet become "self-aware". The program starts by generating a sentence using a recursive traversal through a custom BNF grammar language. Purposely, the questions never quite make complete sense. I wanted to emulate a "being" that was still learning how to communicate.

After the question is generated, I implement Google’s YouTube API to search and play the top search result of the question. I removed all controls from the viewer. So they only thing a viewer can do is watch the presented video or generate a new question and answer with the new button. If the viewer watches a video to completion, the program auto generates a new question and answer.
