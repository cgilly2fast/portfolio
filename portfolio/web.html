Show Myself Off

Although it may seem a little odd to recursively talk about my web portfolio in my web portfolio, this is a website I am proud of. When I first started this project I only had experience with building basic sites. I am hosting this site through GitHub because it is a static site. You can see all my code here.

It also gave me great reps and deepen my skills in Javascript, HTML, CSS. I used this project to teach myself Git. This has added a little bit of time up front but it has saved me many headaches mid-development. Usually, it saves my butt, when I try and add a new feature but break everything in the process. I also got to exercise some of my design skills and employing my schooling in the Photomedia program. Most of the pictures and assets need heavy editing to look just the way I wanted them to on the site.

My proudest improvement to date is removing the contact page and implementing a button that copies my email address to your clipboard. Give it a try and send me an email to say what's up!
