Matuto

Web Art Installation October 2016

Matuto, which means learning in portuguese, is a pseudo-learning software that presents the viewer with a series of statements and concludes with a riddle question. Try it out here.

The learning software

How Matuto works is that, a set of places from UWâ€™s art building are assigned to have a number of artworks. The number of artworks a place has changes each time the game is played. Also how many places that are displayed for each play changes as well. After the riddle is done setting up, Matuto will ask you a question about the places and there artworks. If you are answering correctly you are learning! Incorrectly, well, maybe you should go back to school. View my ARTSgame.js code here.

The final porduct

Matuto as a website was the bulk of the work for this project. However, for the presentation of the website, I setup a desk and computer in a gallery space. I wanted to emulate the workspace of a student studying in the library who had just gotten up to go to the bathroom. I wanted viewers, without me telling them, to snoop a little bit, sit down, and try and figure out the riddle. I have yet had someone figure out the pattern and I encourage you to try.

What I found is that viewers will try and figure out the riddle. But once they realize Matuto wants a non-obvious answer, they would give up and start perusing the files and tabs of my computer. This was a curious and unexpected outcome; maybe I can push this aspect of the project for new piece that deals with voyeurism into a strangerâ€™s computer.
